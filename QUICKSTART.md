# 🚀 Quick Start Guide

## Step 1: Add Your Profile Picture
Place your profile image in the `assets/` folder and name it `profile.jpg`

## Step 2: Customize Your Information
Open `index.html` and find these sections to update:

### Line 45: Your Name
```html
<h1 class="hero-title">Hi, I'm <span class="highlight">Your Name</span></h1>
```
Replace `Your Name` with your actual name.

### Line 46: Your Role/Title
```html
<p class="hero-subtitle">Computer Science Student | Aspiring Data Scientist</p>
```
Update with your role and specialization.

### Lines 47-49: Your Value Statement
```html
<p class="hero-description">
    Passionate about building innovative solutions through technology and data. 
    Let's create something extraordinary together.
</p>
```
Write 2 lines about your passion and what you do.

## Step 3: Update Social Links
Scroll down to lines 102-117 and update these social media URLs:

```html
<!-- GitHub -->
<a href="https://github.com/yourusername" class="social-link" title="GitHub">

<!-- LinkedIn -->
<a href="https://linkedin.com/in/yourusername" class="social-link" title="LinkedIn">

<!-- Twitter -->
<a href="https://twitter.com/yourusername" class="social-link" title="Twitter">
```

## Step 4: Test Your Site
1. Open `index.html` in your browser
2. Test all navigation links
3. Check responsiveness by resizing your browser
4. Test on mobile devices

## Step 5: Customize Colors (Optional)
Open `css/styles.css` and modify these CSS variables:

```css
:root {
    --primary-color: #6366f1;      /* Change this to your brand color */
    --secondary-color: #ec4899;    /* Change this to your accent color */
}
```
